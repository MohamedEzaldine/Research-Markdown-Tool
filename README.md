# 📚 المحول البحثي الاحترافي (Pro Academic PDF to MD)

![Python](https://img.shields.io/badge/Python-3.11+-blue.svg)
![PyQt5](https://img.shields.io/badge/GUI-PyQt5-green.svg)
![License](https://img.shields.io/badge/License-MIT-orange.svg)

أداة متقدمة مفتوحة المصدر، مصممة خصيصاً للباحثين، الأكاديميين، والطلاب، لتحويل الأوراق العلمية والبحثية من صيغة **PDF** إلى صيغة **Markdown** نظيفة ومرتبة. تعتمد الأداة على أحدث نماذج الذكاء الاصطناعي لاستخراج البيانات المعقدة التي تفشل فيها الأدوات التقليدية.

## ✨ الميزات الرئيسية

- **🧠 ذكاء اصطناعي مخصص للأبحاث:** قدرة فائقة على التعرف على المعادلات الرياضية المعقدة وتحويلها إلى كود (LaTeX) سليم.
- **📊 استخراج الجداول المعقدة:** تحليل الجداول العلمية المتداخلة وإعادة بنائها بتنسيق Markdown بدقة عالية.
- **🛡️ خصوصية تامة (Offline-First):** تعمل الأداة محلياً بالكامل على جهازك دون الحاجة للاتصال بالإنترنت (بعد التثبيت الأول)، مما يضمن سرية أبحاثك وبياناتك.
- **🖥️ واجهة مستخدم رسومية (GUI):** واجهة عصرية وسهلة الاستخدام تغنيك عن تعقيدات سطر الأوامر (CLI).
- **📄 دعم التنسيقات الأكاديمية:** تتعامل بسلاسة مع الأوراق البحثية ذات العمودين (Two-column layout) وتستبعد الهوامش غير الضرورية.

## 🚀 كيفية التثبيت والاستخدام

نوفر طريقتين لاستخدام الأداة لضمان أقصى درجات السهولة والشفافية:

### 1️⃣ للمستخدمين (الطريقة الأسهل)
إذا كنت تريد فقط استخدام البرنامج دون الدخول في تفاصيل برمجية:
1. اذهب إلى قسم **[Releases]** (الإصدارات) في يمين هذه الصفحة.
2. قم بتحميل ملف التثبيت الأحدث (مثلاً `Research-Markdown-Tool.exe`).
3. قم بتشغيل الملف واتبع خطوات التثبيت (ملاحظة: سيقوم البرنامج تلقائياً بتحميل نماذج الذكاء الاصطناعي اللازمة لمرة واحدة فقط أثناء التثبيت).
4. افتح البرنامج من سطح المكتب، اختر ملف الـ PDF الخاص بك، وابدأ التحويل!
