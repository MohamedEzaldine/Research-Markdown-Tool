# المحوّل البحثي (Research Markdown Tool) 📄➡️💻

أداة مجانية ومفتوحة المصدر مصممة خصيصاً للباحثين والأكاديميين، لتحويل الأوراق العلمية (PDF) إلى نصوص (Markdown) نظيفة وعالية الدقة. الأداة تعتمد على محرك ذكاء اصطناعي قوي للحفاظ على المعادلات الرياضية، الجداول المعقدة، والتخطيط الأكاديمي.

🌐 **رابط الواجهة المباشر:** [اضغط هنا لتجربة الأداة](https://mohamedezaldine.github.io/Research-Markdown-Tool/)

---

## ✨ المميزات الأساسية
* **معادلات LaTeX:** التعرف على المعادلات الرياضية وتحويلها إلى كود LaTeX سليم.
* **الجداول المعقدة:** إعادة بناء الجداول العلمية بشكل مرتب يحافظ على الصفوف والأعمدة.
* **تخطيط العمودين:** قراءة الأوراق ذات العمودين بالترتيب الصحيح، مع تجاهل الهوامش وأرقام الصفحات.
* **وضعان للتحويل:** "الوضع السريع" لاستخراج النصوص العادية، و"الوضع الدقيق" للمعادلات والجداول.

---

## 🛠️ كيف يعمل المشروع؟ (بنية النظام)
المشروع مقسم إلى جزأين يعملان معاً:
1. **الواجهة الأمامية (Frontend):** ملف `index.html` المرفوع على GitHub Pages (يعمل فوراً للمستخدمين).
2. **الخادم الخلفي (Backend):** ملف `app.py` المبني بـ FastAPI ومحرك `marker-pdf`، وهو المسؤول عن "الوضع الدقيق" ومعالجة الذكاء الاصطناعي الثقيلة.
